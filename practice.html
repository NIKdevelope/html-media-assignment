Q5.What is th: DOCTYPE D:claration in HTML"

Ans. The DOCTYPE Declaration in HTML (Document Type Declaration) is a special instruction placed at the beginning of an HTML document to define its document type and version. It informs the web browser about the rules and specifications to follow when rendering the HTML content. The DOCTYPE Declaration helps ensure that a web page is displayed correctly and consistently across different web browsers, as it tells the browser which version of HTML or XHTML the document adheres to.
